# Demo Video Script

Target length: 4 minutes 30 seconds. Maximum contest length: 5 minutes.

## 0:00-0:25 - Problem

"Travel apps help us decide where to go next. Reality Archive remembers the places that already shaped us. It turns an old cafe, a street, or a landmark into a personal living museum."

Show the Home page and the map.

## 0:25-1:05 - Find Or Add A Place

"I can discover nearby places with Geoapify and OpenStreetMap, search the clearly labeled Iloilo demo archive, or add a place manually. Manual places work locally even when a map API is unavailable."

Open Explore. Point out the **Demo archive** label, then manually add a genuine place meaningful to you.

## 1:05-2:00 - Preserve A Memory

"I will preserve a new memory for this place. Reality Archive supports text, a local photo, or a two-minute voice recording. The place and memory stay in this browser."

Save one genuine memory. Avoid private information. Open the Museum and show that the new place appears.

## 2:00-2:40 - Local-First Museum

"The Museum is driven by places I actually saved. If there are no user places, it stays empty instead of inventing exhibits. A local grounded story remains available without Azure."

Open the place exhibit and show the saved memory and local story.

## 2:40-3:40 - Foundry IQ

"Cloud curation is optional. Before anything is sent, Reality Archive explains exactly what data will go to Microsoft Azure. Photo files and recorded audio remain local."

Check the consent box and enter the private demo access code off camera or with the field obscured.

"The protected Next.js route uploads normalized archive documents to Azure AI Search. A managed Foundry IQ knowledge base retrieves grounded passages and references. A Microsoft Foundry agent or model then writes the museum story using only that evidence."

Generate the live story and show the **Live Foundry IQ story** status and citations. Do not film this step unless the real knowledge base has been successfully verified.

## 3:40-4:15 - Architecture And Safety

Show `docs/architecture.svg`.

"Azure routes require a signed, expiring session, exact origin checks, explicit consent, request-size limits, and rate limits. The app never exposes Azure keys in the browser and never labels a fallback as live Foundry IQ."

## 4:15-4:30 - Close

"Reality Archive is a private, grounded way to preserve the geography of a life. Places do not only hold coordinates. They hold us."
